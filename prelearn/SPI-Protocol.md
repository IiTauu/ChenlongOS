# SPI协议
---

SPI（串行外设接口）是高速、全双工、同步的通信总线，有以下特征
- 一主多从式架构
- 时钟、主机输入、主机输出、片选四根信号线
- 基于移位寄存器实现主从设备数据环形交换
- 通过时钟极性和时钟相位组合成四种工作模式

## 信号线组成

串行时钟SCLK。时钟信号由主机产生。设备在时钟信号边沿采样信号。

主机输出MOSI。主机输入MISO。

片选CS/SS。主机通过该线指明通信的从设备。

主从设备之间共享除片选线以外的所有线。每个从设备都与主机有一条专用的片选线。

## 数据传输原理

SPI的通信本质上是两设备中移位寄存器的数据的交换，也即发送一个字节必然会接收一个字节。

主机只需要发送时，会忽略接收到的数据；只需要接收时必须先发送一个空数据产生时钟信号，驱动从设备输出

在移位寄存器中最高有效位先行发送。

## 工作模式

时钟极性CPOL决定SCLK的空闲时电平，置0时空闲电平为低电平；置1时为高电平。

时钟相位CPHA决定设备在波的哪一边沿采样，置0时在前边沿采样；置1时在后边沿采样。

| 模式  | CPOL | CPHA | 特性     |
| --- | ---- | ---- | ------ |
| 0   | 0    | 0    | 空低，上升沿 |
| 1   | 0    | 1    | 空低，下降沿 |
| 2   | 1    | 0    | 空高，下降沿 |
| 3   | 1    | 1    | 空高，上升沿 |
